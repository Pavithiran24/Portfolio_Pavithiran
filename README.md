# Student Portfolio - Thevarasa Pavithiran (PPW)

Modern, professional student portfolio built with React + Vite for a university PPW assignment submission.

## Run locally

1. Install dependencies
   - `npm install`

2. Start dev server
   - `npm run dev`

3. Build for production
   - `npm run build`

4. Preview production build
   - `npm run preview`

## PDF export (recommended)

Use the browser print dialog:

- Open the site in the browser.
- Press `Ctrl + P` → Destination: **Save as PDF**.
- Enable **Background graphics** for best results.

## Editing

- Portfolio content: `src/App.jsx`
- Styling/theme: `src/index.css`
